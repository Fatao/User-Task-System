

# User Authentication & Task Management System

## Overview

This repository contains the code for a **User Authentication and Task Management System** built in **Python**. It is a simple but functional application designed to manage users and their associated tasks efficiently.

The system allows users to securely register and log in, manage their personal profile details, and perform full CRUD (Create, Read, Update, Delete) operations on their tasks.

  * **Security:** Enforces password validation (minimum 6 characters) to ensure basic security.
  * **Profile Management:** Users can view and update their profile information after logging in.

  * **Delete Tasks:** Remove completed or unwanted tasks by selecting the task number.
  * **Task Assignment:** The system supports associating specific tasks with users.
